# Ethan A. Caspian personal site

Netlify-ready Vite + React + Tailwind project bundle.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm install
npm run build
```

## Netlify deploy

Upload this whole folder to Netlify, or connect it as a Git repository.

Build settings are already included in `netlify.toml`:

- Build command: `npm run build`
- Publish directory: `dist`
